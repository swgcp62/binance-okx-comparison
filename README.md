## 보안과 자산 안전성: 어디가 더 믿을 만한가

보안은 거래소를 고를 때 수수료만큼, 아니 그 이상으로 중요한 기준이다.

- **바이낸스 SAFU**: 바이낸스는 사용자 보호 기금인 SAFU(Secure Asset Fund for Users)를 운영하며, 규모는 약 10억 달러에 달한다. 해킹이나 내부 오류 등으로 자산 손실이 발생했을 때 보상 재원으로 활용된다.
- **OKX PoR(자산 증명)**: OKX는 월간 자산 증명(Proof of Reserves) 보고서를 공개한다. 유저의 예치 자산이 1:1 이상으로 실제 보유돼 있음을 독립 감사를 통해 입증하는 방식이다. 별도의 보험 기금도 운영 중이다.

두 거래소 모두 탑티어 보안 인프라를 갖추고 있으며, 과거 크고 작은 사건을 겪으면서 오히려 더 강화된 케이스다. 현재 기준으로는 두 거래소 모두 안전 측면에서 신뢰할 수 있는 수준이다.
